This repository contains the data processing and visualization scripts used in the study: 

*"Functional diversity among crop models reveals contrasting resilience of soybean and maize to heatwaves and CO₂ enrichment"*

To replicate the figures, download the simulation dataset from Zenodo https://doi.org/10.5281/zenodo.19095107 and extract it into the /data folder of this repository.

* The map figures in the Jupyter notebook are, in some cases, slightly different from those in the article because the article figures were further edited after the notebook generated the output images.
* For example, in Figure 1, the county names and agricultural districts were added after the image was produced by the code.
